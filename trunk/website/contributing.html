Contributing

Join the development mailing list if you would like to contribute to the book. you can subscribe to this list on the following page:

http://libranet-basics.org/mailman/listinfo/devel_libranet-basics.org

Read the "Using SVN" HOWTO for information about committing patches to the Libranet Basics SVN tree.
